# Connector Nodes  

Connector nodes consume the processed messages from Kafka topics 
and send them to the database.  

## Operation  

To connect the messages from the topics to the database, run  
    
```shell script
sh ~/admorphous/start_connectors.sh
```

## Setup    

OS: Linux  

Install:  
- Java  
- python  
- pip  
- Kafka    
- git
